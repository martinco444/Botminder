# Botminder
Bot público de Telegram para programar recordatorios. 

## Cómo usarlo
1. Abre Telegram y busca el bot por su username: `@BotminderBot`  
   o usa el enlace directo: `https://t.me/BotminderBot` (reemplaza por el username real).
2. Envía `/start`.
3. Comandos principales:
   - `/recordatorio` — iniciar flujo para crear un recordatorio.
   - `/ver` — listar tus recordatorios pendientes (si está habilitado).
4. El bot te pedirá mensaje y fecha/hora en formato `YYYY-MM-DD HH:MM`.

## Privacidad y datos
- Los recordatorios se guardan en una base de datos gestionada por el autor.

## Notas para usuarios
- No necesitas clonar ni ejecutar este repositorio para usar el bot.
- Si quieres reportar errores o solicitar mejoras, abre un issue o PR en el repositorio.

## Para desarrolladores (opcional)
- Este repositorio contiene el código fuente. Si deseas ejecutar tu propia instancia (solo si eres desarrollador), revisa el código y la sección de configuración; ten en cuenta que la instancia pública y su token se mantienen privadas.
